#  Documentación de Markdown

#  ¿Qué es Markdown?

Markdown es un lenguaje de marcado ligero que permite dar formato al texto de una manera sencilla usando símbolos especiales.

---

#  Encabezados

### ✔ Cómo se escribe:
```markdown
# Título H1
## Título H2
### Título H3
```

### ✔ Cómo se ve:
# Título H1
## Título H2
### Título H3

---

#  Negritas, cursivas y tachado

### ✔ Cómo se escribe:
```markdown
**Negritas**
*Cursivas*
***Negritas y cursivas***
~~Tachado~~
```

### ✔ Cómo se ve:
**Negritas**  
*Cursivas*  
***Negritas y cursivas***  
~~Tachado~~

---

#  Listas
